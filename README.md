## 📦 安装与使用

### 系统要求
- Node.js >= 18.0.0
- macOS / Linux / Windows
- Git (用于版本管理)

### 安装
```bash
# 克隆仓库
git clone https://github.com/yun520-1/mark-heartflow-claw.git
cd mark-heartflow-claw

# 安装依赖
npm install

# 全局安装 (可选)
npm link
```

### 基本使用
```bash
# 查看状态
heartflow status

# 运行测试
heartflow test

# 手动升级
node scripts/auto-upgrade.js

# 启动 API 服务器
heartflow-api
```

### 自动升级
HeartFlow 支持 23 分钟自动升级循环：
```bash
# 配置 cron (macOS/Linux)
crontab -e
# 添加：*/23 * * * * cd ~/mark-heartflow-claw && node scripts/auto-upgrade.js
```
